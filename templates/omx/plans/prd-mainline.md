# PRD - {{DISPLAY_NAME}} Mainline

## Product Goal

把 `{{DISPLAY_NAME}}` 推进成一条长期可持续推进的 active mainline，而不是停留在碎片化任务推进。

## Current Program Scope

当前 mainline 至少要冻结：

- north star
- 当前阶段与 long-horizon order
- 明确不做的范围
- 最小可执行基线

## Required Decisions

在正式长跑前，这份 PRD 至少应补齐：

1. 当前 north star 是什么
2. 当前唯一活跃子线是什么
3. 下一条最小可执行基线是什么
4. 哪些 future surfaces 当前只是 guardrail，不是实现目标

## Non-Goals

如果某项工作不服务于当前 active mainline，就不应默认进入本轮 OMX 长跑。
